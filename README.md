This is a microservice project using SpringBoot V3.3.2

Used:-

    1. Config server
    2. Cinfig Client
    3. Web Client - Rest Template
    4. LoadBalancer
    5. API Gateway
    6. Discovery Service
    7. Fault Tolerance using Resilience4j
    8. Resilience4j - CircuitBreaker, Retry, RateLimiter
